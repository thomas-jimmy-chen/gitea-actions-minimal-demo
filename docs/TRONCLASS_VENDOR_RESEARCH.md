## 六、TronClass 特點

1. **雲端 SaaS 架構** - 支援多租戶部署
2. **多終端支援** - Web、iOS、Android
3. **API 整合** - RESTful API 設計
4. **功能模組**:
   - 課程管理
   - 考試系統
   - 學習時數追蹤
   - 公告系統
   - 學習履歷統計

## 九、EEBot 相關技術發現

根據 codebase 分析，TronClass API 端點特徵：

```
/api/courses/{id}/activities    # 課程活動
/api/courses/{id}/exams         # 考試列表
/api/my-courses                 # 我的課程
/api/statistics/user-visits     # 學習時數統計
/api/announcement               # 公告系統
```

### 典型 VIE 架構

這是典型的中國科技公司境外上市/融資架構：
1. **開曼控股公司** - 便於國際融資
2. **境內運營實體** - 實際開發與運營
3. **港台子公司** - 拓展海外市場
